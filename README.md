# Customer Behavior & Campaign Performance Dashboard

## Overview
This project analyzes customer purchasing behavior and campaign effectiveness using Power BI.

The dashboard provides insights into:
- Customer spending patterns
- Purchase channel distribution
- Campaign response behavior
- Key performance indicators

## Key Insights
- Store purchases dominate overall customer activity
- Graduates show the highest campaign response rate
- High spending customers are more likely to engage with campaigns

## Tools Used
- Power BI
- Excel

## Features
- Interactive dashboard with slicers
- KPI cards for quick overview
- Multiple visualizations (bar, line, column, table)
- One-page structured layout

## Files
- Dataset: marketing_campaign.xlsx
- Dashboard: Customer_Analytics.pbix
- Preview: dashboard.png

## How to Use
1. Open the .pbix file in Power BI Desktop
2. Use slicers to filter customer segments
3. Explore insights across different visuals
